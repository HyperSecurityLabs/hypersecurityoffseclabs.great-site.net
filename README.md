> **↳ Portal** — The primary gateway into the HyperSecurity Offensive Labs ecosystem. Serves as the central hub for tooling, research publications, lab environments, and offensive security resources. All operational threads converge here — from framework downloads to real-time intelligence feeds, making it the single point of entry for operators and researchers alike.

> **↳ Forums** — A community-driven discussion board built for security researchers, developers, and red-team operators. Covers topics ranging from vulnerability disclosure and exploit development to tooling collaboration and operational tradecraft. Designed to foster knowledge exchange across the offensive security community.

> **↳ Knowledge** — A curated library of technical write-ups, in-depth guides, and reference materials covering penetration testing methodologies, reverse engineering, privilege escalation chains, and everything in between. Continuously updated with field-tested techniques from real-world engagements.

> **↳ Kronik** — A WordPress attack and assessment framework for penetration testing. Enumerates vulnerable plugins, misconfigured themes, user roles, and version-based weaknesses via automated scanning. Supports post-exploitation workflows including backdoor deployment, credential extraction, and privilege escalation against WordPress deployments.

> **↳ OXIDE** — AI-augmented web vulnerability scanner written in Rust. Features 15+ concurrent scanners covering SQLi, XSS, LFI, CMDi, CORS misconfig, TLS weaknesses, and credential leaks. Includes a WAF bypass engine, ML-driven zero-day anomaly detection via Random Forest & SVM, async web crawler, and an encrypted test database of known exploits.

> **↳ OXIDE Elite** — Dedicated discussion group for OXIDE Framework operators and developers. Covers scanner configurations, custom payload development, WAF bypass techniques, and collaborative research around the OXIDE ecosystem.

> **↳ Telegram** — Official channel for framework release announcements, zero-day disclosures, and curated threat intelligence from the HSOL ecosystem. Delivers real-time updates on new tooling, research publications, and community-driven security operations.
